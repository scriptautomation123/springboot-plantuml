Markdown PlantUML Processor

Upload a markdown file containing PlantUML blocks to generate processed content with diagram images.

How it works:
  1. Upload a markdown file containing PlantUML code blocks
  2. The server will process each ```plantuml block
  3. Generate SVG/PNG images for each diagram
  4. Replace code blocks with image links
  5. Download a ZIP file containing the processed markdown and images
